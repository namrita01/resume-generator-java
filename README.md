**ResuGen** is an intelligent resume generator that allows users to create highly customizable and ATS-optimized resumes. Built with **JavaFX** for UI, **JDBC** for database management, and integrated with the **OpenAI API** for smart content optimization, it helps users generate job-ready resumes quickly and efficiently.

---

## 🚀 Features

- 🔐 **User Authentication** (Admin & User Mode)
- 🎨 **Resume Template Selection**
- 📄 **Auto-fill Details from Database**
- 🧠 **OpenAI-powered Resume Optimization** (ATS Friendly)
- 📊 **Dashboard** to track usage (resumes created, job roles, etc.)
- 📥 **Export as PDF, Word, or LaTeX**

---

## 🧱 Tech Stack

| Layer         | Technology        |
|--------------|--------------------|
| Frontend     | JavaFX             |
| Backend      | Java               |
| Database     | JDBC               |
| AI API       | OpenAI GPT         |
| Export Tools | iText / Apache POI |

---

## 🖼️ UI Flow

1. **Login Page** → Choose User/Admin
2. **Dashboard** → View previous resumes or create new
3. **Template Selection** → Choose a resume format
4. **Form Fill** → Autofilled or manual
5. **AI Optimization** → Paste job description & optimize
6. **Export** → Download as PDF/Word/LaTeX
